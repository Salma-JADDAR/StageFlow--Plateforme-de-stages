# 🎯 StageFlow - Plateforme de Gestion de Stages

## 📝 Description
StageFlow est une plateforme web innovante qui connecte les étudiants à la recherche de stages avec les entreprises recruteuses. 
Grâce à un système de matching intelligent basé sur l'IA, la plateforme facilite la mise en relation entre les talents et les opportunités professionnelles.

## ✨ Fonctionnalités principales

### Pour les Étudiants
- 🔍 Recherche et filtrage avancé des offres de stage
- 🤖 Recommandations personnalisées basées sur l'IA
- 📄 Gestion du CV et du profil
- 📊 Suivi des candidatures en temps réel
- 🎓 Gestion des formations et compétences
- 🔔 Notifications en temps réel

### Pour les Recruteurs
- 📝 Publication et gestion des offres de stage
- 👥 Gestion des candidatures reçues
- 📊 Tableau de bord avec statistiques
- 🏢 Gestion du profil entreprise
- 🤝 Matching intelligent avec les profils étudiants

### Pour les Administrateurs
- 👤 Gestion des utilisateurs (activation/désactivation/suppression)
- 📋 Validation et modération des offres
- 📈 Tableau de bord avec statistiques globales
- 🔐 Gestion des administrateurs

## 🛠️ Technologies utilisées

### Frontend
- ⚛️ React 18
- 🎨 Tailwind CSS
- 📦 React Router DOM
- 🔥 React Hot Toast
- 📱 Responsive Design

### Backend
- 🐘 Laravel 10
- 🔐 JWT Authentication
- 🗄️ MySQL
- 🤖 Système de matching IA
